# Arey Pagal — a soda lab

Marketing site for **Arey Pagal**, a small soda lab in Markham, ON. Every soda is made in house and brewed on site. The whole menu is two lines — **One Pour** and **Three Pours** — served *dealer's choice*.

**Live site:** https://iamporthos.github.io/cuorepazzo/

---

## What's here

The entire site is a single self-contained file:

- **`index.html`** — the whole website (all pages) in one file, with fonts loaded from Google Fonts and the logo images embedded inline. This is the only file GitHub Pages needs to serve.

Pages live inside `index.html` and switch via URL hashes:

| Page | Link |
|------|------|
| Home | `index.html` or `#home` |
| Menu | `#menu` |
| Story | `#ourstory` |
| Workshop | `#story` |
| Visit | `#contact` |
| Menu card | `#menupdf` |

### Source files (optional, for editing)

`index.html` is **generated** from these editable source files. You don't need them to host the site, but keep them if you want to make changes:

- `index.dc.html` — Home
- `menu.dc.html` — Menu
- `ourstory.dc.html` — Story
- `story.dc.html` — Workshop
- `contact.dc.html` — Visit
- `menu-pdf.dc.html` — printable menu card
- `logo_lockup.png`, `logo_face.png` — logo artwork
- `support.js` — runtime for the source files

## Hosting on GitHub Pages

1. Upload **`index.html`** to the repository root.
2. Go to **Settings → Pages**.
3. Under **Source**, pick **Deploy from a branch** → branch **main** → folder **/ (root)** → **Save**.
4. Wait 1–3 minutes, then visit the URL shown on that Pages screen.

> **Note:** after replacing `index.html`, your browser may cache the old version. Do a hard refresh (Cmd/Ctrl + Shift + R) or open it in a private window to see changes.

## Brand

- **Colors:** forest green `#1B3A2A` · cream `#E8DFB8` · gold `#AC903D` · mint `#80ffa6`
- **Contrast rule:** dark on light, or light on dark — never mid-tone on mid-tone.
- **Type:** Shrikhand (wordmark) · Anton (headings) · Hanken Grotesk (body)
- **Logo:** dark linework only — never inverted. On dark sections it sits inside a cream or gold badge.
